# Шрифты

Положи сюда свой файл шрифта в формате `.ttf` или `.otf`

## Инструкция:

1. Скопируй свой шрифт в эту папку
2. Переименуй его в `YourFont.ttf` (или любое имя)
3. Обнови `FONT_PATH` в `badge_bot.py`:
   ```python
   FONT_PATH = "fonts/ИмяТвоегоШрифта.ttf"
   ```

## Где найти шрифты:

- [Google Fonts](https://fonts.google.com/) - бесплатные шрифты
- [DaFont](https://www.dafont.com/) - огромная коллекция
- [Font Squirrel](https://www.fontsquirrel.com/) - бесплатные для коммерции

## Формат:

- ✅ `.ttf` (TrueType Font)
- ✅ `.otf` (OpenType Font)
- ❌ `.woff`, `.woff2` (веб-форматы, не поддерживаются)
